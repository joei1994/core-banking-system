# Core Banking System

A basic microservice-based core banking system built with Spring Boot and Docker.

## Tech Stack

- Java 17
- Spring Boot
- Maven
- Docker

### Prerequisites

- JDK 17
- Docker
- Maven

### How to start

```
git clone https://github.com/joei1994/core-banking-system.git
```

### Run the project

```bash
docker-compose up
```
